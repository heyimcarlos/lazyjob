Stats Module

DashboardStats

pub struct DashboardStats {
    pub total_jobs: i64,
    pub applied_this_week: i64,
    pub in_pipeline: i64,        // non-terminal, non-Interested
    pub interviewing: i64,       // PhoneScreen + Technical + Onsite
    pub stage_counts: HashMap<ApplicationStage, i64>,
}

SQL Queries

Total Jobs

SELECT COUNT(*) FROM jobs

Applied This Week

SELECT COUNT(*) FROM applications
WHERE stage != 'interested'
AND updated_at >= now() - interval '7 days'

Per-Stage Counts

SELECT stage, COUNT(*) FROM applications GROUP BY stage

The in_pipeline and interviewing counts are computed from the stage breakdown in Rust, avoiding extra queries.

Stale Applications

SELECT a.id, j.title, j.company_name,
       (EXTRACT(EPOCH FROM (now() - a.updated_at)) / 86400.0)::FLOAT8
FROM applications a JOIN jobs j ON a.job_id = j.id
WHERE a.stage NOT IN ('accepted', 'rejected', 'withdrawn')
AND a.updated_at < now() - interval '14 days'
ORDER BY a.updated_at ASC

Note: ::FLOAT8 cast is required because PostgreSQL's EXTRACT(EPOCH) returns NUMERIC, not FLOAT8.

App Wiring

New Method: App::load_dashboard_stats()

pub async fn load_dashboard_stats(&mut self) {
    let Some(pool) = &self.pool else { return };
    let stats = match stats::compute_dashboard_stats(pool).await {
        Ok(s) => s,
        Err(err) => {
            tracing::warn!("Failed to load dashboard stats: {err}");
            return;
        }
    };
    let stale = match stats::find_stale_applications(pool).await {
        Ok(s) => s,
        Err(err) => {
            tracing::warn!("Failed to load stale applications: {err}");
            Vec::new()
        }
    };
    self.views.dashboard.set_stats(stats, stale);
}

Call Sites

Features

  • 4 stat blocks with color-coded values (blue, green, yellow, magenta)
  • 9-column pipeline showing per-stage counts with stage-appropriate colors
  • Scrollable stale application list with warning icons and days-stale counter
  • Selected item highlighted with theme's selected style (white on blue, bold)
  • Graceful empty state: "No stale applications. You're on top of things!"
  • Loads on startup and refreshes with Ctrl+R
