Task 19 · spawn()

spawn() — Subprocess Launch

pub async fn spawn(
    &mut self,
    loop_type: &str,
    params: serde_json::Value,
) -> Result<RunId> {
    let run_id = RunId::new();

    // 1. Build command: binary [extra_args...] worker
    let mut cmd = Command::new(&self.binary_path);
    for arg in &self.binary_args { cmd.arg(arg); }
    let mut child = cmd
        .arg("worker")
        .stdin(Stdio::piped())
        .stdout(Stdio::piped())
        .stderr(Stdio::null())
        .spawn()?;

    // 2. Take handles before moving child
    let mut stdin = child.stdin.take().expect("stdin piped");
    let stdout = child.stdout.take().expect("stdout piped");

    // 3. Send Start command
    let encoded = NdjsonCodec::encode(&WorkerCommand::Start {
        loop_type: loop_type.to_string(), params
    });
    stdin.write_all(encoded.as_bytes()).await?;
    stdin.flush().await?;

    // 4. Background reader task → broadcast
    let event_tx = self.event_tx.clone();
    tokio::spawn(async move {
        let reader = BufReader::new(stdout);
        let mut lines = reader.lines();
        while let Ok(Some(line)) = lines.next_line().await {
            if let Ok(event) = NdjsonCodec::decode(&line) {
                let _ = event_tx.send((run_id, event));
            }
        }
    });

    self.running.insert(run_id, ProcessHandle { child, stdin });
    Ok(run_id)
}
Task 19 · cancel()

cancel() — Graceful Shutdown

pub async fn cancel(&mut self, run_id: &RunId) -> Result<()> {
    let handle = self.running
        .get_mut(run_id)
        .ok_or_else(|| RalphError::NotFound(run_id.to_string()))?;

    // Step 1: graceful — send Cancel to stdin
    let cancel_line = NdjsonCodec::encode(&WorkerCommand::Cancel);
    let _ = handle.stdin.write_all(cancel_line.as_bytes()).await;
    let _ = handle.stdin.flush().await;

    // Step 2: wait up to 3s for graceful exit
    let wait = tokio::time::timeout(
        Duration::from_secs(3),
        handle.child.wait(),
    ).await;

    // Step 3: SIGKILL if timeout
    if wait.is_err() {
        let _ = handle.child.kill().await;
    }

    self.running.remove(run_id);
    Ok(())
}

Cancel flow

1. Send WorkerCommand::Cancel to stdin
↓
2. tokio::time::timeout(3s, child.wait())
↓
3a. OK → process exited gracefully
or
3b. Timeout → child.kill() (SIGKILL)
↓
4. running.remove(run_id)

Error handling

Write errors to stdin are silently ignored — the subprocess may have already exited before cancel is called.

Cancelling an unknown RunId returns RalphError::NotFound immediately.

// Only hard error: unknown run
Err(RalphError::NotFound(id))
Task 19 · Testing

Test Strategy

The key challenge: testing subprocess spawning without a real ralph binary. Solution: with_binary_and_args() + sh -c.

Why not temp executable files?

Initial approach used tempfile::NamedTempFile + chmod 755. Under parallel async test execution, Linux raises ETXTBSY ("Text file busy") — a file being written and executed concurrently.

Fix: use sh -c '...' as the "binary" with the script inlined as an arg. No temp files, no OS-level conflicts.

Task 19 · Key Decisions

Key Design Decisions

broadcast channel type: (RunId, WorkerEvent)

Rather than wrapping events in a separate RalphEvent enum (as the spec sketches), we broadcast tuples. This keeps the broadcast channel agnostic of the wrapper enum while still letting every subscriber know which subprocess produced each event.

with_binary_and_args() — testability by composition

binary_args: Vec<OsString> injected before the hardcoded "worker" arg. Production: binary_args=[], binary_path=current_exe(). Tests: binary_args=["-c", "..."], binary_path="sh". Zero test-specific conditionals in production code.

Background reader task: best-effort decode

Decode errors from subprocess stdout are silently dropped. A misbehaving subprocess should not crash the TUI — the background task continues reading until EOF. This is intentional.

ChildStdin taken immediately on spawn

child.stdin.take() is called before sending the Start command. If this were deferred, child would need to be mutably borrowed while also being moved into ProcessHandle. Taking early avoids the borrow conflict.
